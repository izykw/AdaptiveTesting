# Adaptive testing
Adaptive testing - testing, where each next question is selected automatically, based on the answers given to previous questions, and a predetermined level of complexity.

# Technology stack (Frontend)
<ul>
<li>React</li>
<li>React Router</li>
<li>Bootstrap</li>
</ul>
